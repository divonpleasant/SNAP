FORUM Reference Index

Description

FORUM is software for managing, archiving, and viewing patient data, images, videos, and reports provided by diagnostic devices or other documentation systems. Different documents can be flexibly arranged side by side and easily compared with each other and viewed across connected computers using FORUM Viewer.

FORUM enables the customer to centrally schedule patient examinations and forward these to connected devices as Modality Worklist Orders (MWL). The MWL contains all patient demographics, so the data does not need to be entered manually on the device. Once captured, images will be sent back to FORUM to become part of the patient record.

FORUM manages DICOM files. A DICOM file contains both image data from the patient master data (demographics). FORUM reads in the patient master data from the DICOM files and stores them together with a reference to the image data ("documents") in the FORUM database and saved in the storage location.

FORUM can work with all devices and software products that support the DICOM standard and which are compatible with FORUM in accordance with our DICOM Conformance Statement.

FORUM can be used as the primary image and report management system, but can also be integrated into an existing system via interfaces. Patient master data is then managed by the leading system.
