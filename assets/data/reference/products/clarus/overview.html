System Description

Clarus® is a non-contact, high-resolution imaging device for in vivo imaging of the human eye. Imaging modes include:

  1. True color reflectance imaging
  2. FSF-Blue (Fundus autoflorescence with blue excitation)
  3. FAF-Green (Fundus autoflorescence with green excitation)
  4. Flourescein angiography (FA) — Model 700 only
  5. Stereo imaging
  6. External eye imaging

Using a monochromatic camera, a broad line of illumination scans across the entire retina.

Red, green, and blue Light Emitting Diodes (LEDs) sequentially illuminate to generate true color images. The broad bandwidth of LEDs gives a true color rendering. Blue and green LED illumination enables Fundus autoflourescence (FAF) imaging and Flourescein angiography (FA) imaging (700).

By illuminating only a narrow strip of the retina at a time, the illumination stays out of the viewing path, keeping haze and flourescence of the anterior segment out of the retinal image and allowing a clear view of much more of the retina than the annular-ring illumination used in traditional fundus cameras (e.g. VisuCam).

Field of View Options

  • Widefield: A single image with a 133° FOV when measured from the center of the eye (90° when measured as described in ISO 10940)
  • Ultra-widefield: Two images stitched together into a montage with a 200° FOV when measured from the center of the eye
  • Auto Montage: A 4-scan montage using preset fixation targets
  • Montage: 2-6 widefield images stitched into a single, combined retina image

Additional Options

  • Stereo imaging
  • External eye imaging

Software Overview

All instruments and review stations on the network must run the same software version.

The Clarus 700 system ships with the software application installed. The same application is used for the Review Software.

Anti-virus software could conflict with system installation files, causing installation problems or preventing the software from launching. Common conflicts include:

  • Misidentifying installation files as a possible threat
  • Quarantining required installation files

Before Installing the Software

Configure anti-virus software to exclude the following folders from on-access and on-demand scanning:

  • C:\Program Files\CZM\CLARUS
  • C:\ProgramData\Carl Zeiss Meditec\Review (review software only)

Suspend anti-virus detection during installation procedure.

Review Software

Review software is installed on computers that clinicians use to compare, analyze, annotate, and save scans acquired with the Clarus.

Common Actions in Clarus Software

Importing Patient Records

When patient records are imported from an EMR, they are reconciled with local records as follows:

  • Same ID and Issuer of ID, different name, date of birth, or gender: the local record is updated to match the EMR record
  • Different ID or Issuer of ID: a new patient record is generated

Some EMR systems allow records that are missing key identifying information. Clarus automatically adds the default "Unknown" if the name is missing but rejects records that do not have a Patient ID or date of birth.

Updating Windows Enhanced Security Settings

Refer to Clarus Service Bulletin 055 for detailed information.

Instruments with the following serial numbers were manufactured with the incorrect Windows enhanced security settings:

  • Clarus 500: CL500-53000 through CL500-54530
  • Clarus 700: CL700-71908 through CL700-73321

For Clarus instruments that were not properly configured during manufacturing, an obvious difference is that the Windows login screen defaults to the Tech Support user.

After updating Windows enhanced security per the instructions below, the Windows login screen defaults to the most recent user.

Adding a Patient

Name, date of birth (DOB), and gender are required fields when adding a new patient. Click the related field to enter information.

Manually Correcting Laterality

You can change the laterality of an image after it is saved using the Review or Proof screens.

Manual laterality change is not possible if:

  • After the data is exported
  • In connected mode (FORUM)
  • On imported data

Deleting a Patient Record

See IFU for detailed instructions.

Patient records are deleted via the Delete sub-menu.

  • Patients deleted on the instrument are not deleted from the EMR system

Merging Patient Records

See IFU for detailed instructions.

If two records are inadvertently created for the same patient, they can be merged via the Merge sub-menu.

Select the duplicate patient record(s).

  • You can merge up to twenty (20) patient records
  • Select Compare to view patients side by side

Select Merge.

Enter a reason for the merge in the text box, select Merge. The exam data from the duplicate patient records is added to the original patient record.

Reassigning Image Scan

See IFU for detailed instructions.

Moving incorrect data from one patient to the correct patient using the Tests sub-menu.

Click Reassign.

Select correct patient, click Reassign.

Enter reason for the reassignment in the text box, click Reassign.

System Details

Calibration File Transfer

If there is a system message about calibration file cross check or calibration file mismatch, try a calibration file transfer. Try Device to Host first, as the file on the microcontroller is less likely to be corrupt.

  • Done via Calibration Wizard. Launch Wizard using icon on Windows Desktop
  • Must select a Tool and Power Head before Calibration File Transfer will become available to select (SW 1.1.X or greater)
  • Close Calibration Wizard when finished or it may cause an application failure

Safety and Certifications

Compliance

Clarus 700 instruments are compliant with ISO 10940:2009 and ISO 15004-1:2006.

Laser and Optical Safety

Clarus 700 is compliant with the following optical safety standards:

  • IEC 60825-1
  • ANSI Z80.36
  • ISO 15004-2

Classification

Group 1 instrument – Per ANSI Z80.36 and ISO 15004-2, Group 1 instruments are ophthalmic instruments for which no potential light hazard exists.

Class 1 laster product – Per IEC 60825-1.
