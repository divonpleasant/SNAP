Device Overview

The Humphrey Field Analyzer (HFA) 3 is a device designed to test a patient's eyes for blind spots (scotomas) in their field of vision. The test is performed by projecting stimuli (spots of light) of varying brightnesses onto the surface of the device's bowl in a specific pattern. The patient responds to the stimulus they can see. The device tests for absolute scotoma (no vision) and relative scotoma (decreased vision).

The HFA3 device should be started in a dimly lit room as it performs brightness checks on boot-up to verify calibration before testing is allowed.

Device Notes

  • The HFA3 includes a touchscreen display with an onscreen keyboard for data entry. An external USB keyboard and mouse can be used for data input and interface navigation

Software Overview

Title Bar

  • Liquid Trial Lens. Appears if the Liquid Trial Lens is installed (model 860 only) and flashes red during lens adjustment. Do not touch the trial lens when the icon is blinking
  • Privacy Mode. Hides the screen from anyone who does not need to view the patient data. Click the Continue button to return to the patient data
  • Help. Opens the on-screen user manual
  • Brightness. Displays a control for adjusting the screen brightness
  • Volume. Displays a slider control to adjust the sound output volume
  • Settings. Opens the Settings window. When there is a new alert, an exclamation mark appears on the Settings icon. Clicking the exclamation mark opens the Message history
  • Close. Opens a menu with options for logging out the current user or shutting down the device

Software Notes

  • HFA Review Software is not compatible with FORUM and cannot be used with an HFA3 instrument connected to FORUM

Printer Support

The HFA3 is compatible with PostScript printers, including shared network printers and wireless printers using a wireless USB adapter, or directly connected using the ethernet port. Direct USB connection is not supported.
