HFA (Humphrey Field Analyzer) Reference Index

Description

The HFA3 is a device used to test the patients visual field for lost or decreased vision.

This is typically a "monocular" test, where one of the patient's eyes is patched during the testing procedure and repeated for each eye. Patient fixation is crucial to the test result and the HFA3 can monitor that in various ways.
