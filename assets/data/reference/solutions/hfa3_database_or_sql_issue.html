The application may display a message saying:

  • Critial Error
  • Emergency Shutdown
  • Application encountered an error

Regardless of DICOM usage, if the logs include the string, Database or SQL issue, this is likely due to some problem with the local MySQL instance or the database on that instance.

Patient Disassociation

If the instrument is connectedo to a DICOM EMR and there is patient disassociation referenced in the logs as well, a new database is likely necessary.
