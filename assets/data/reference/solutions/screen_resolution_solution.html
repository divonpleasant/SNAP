To resolve the error, click OK to close the application, then follow these steps:

  • Go to Windows Settings (use the Taskbar Search and enter "Settings"; it has a gear icon)
  • Select System > Display
  • Scroll down to the section titled, "Scale & Layout"
  • Change the Scale setting from 125% to 100%
  • Make sure the Display Resolution setting is 1920 x 1080 (or higher)
  • Restart the Cirrus HD-OCT application
